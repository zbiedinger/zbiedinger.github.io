Cool Projects

At DevCode Camp I've amassed a stockpile of projects under my belt. Eger to start new projects and design challenges to solve. Here are a few of the applications and projects I've done.


Custom List

This is a custom list of type T with Add(), Remove(), and resizing functionality. The plus '+' and minus '-' operators are overloaded and used to create a Zip() method of two CustomList<T>.

Soda Machine

A Console application representing a Soda Machine that takes in multiple payment methods and dispenses soda.

RPSLS Game

An interactive console application of "Rock, Paper, Scissors, Lizard, Spock." It's a two-player game based on the classic "Rock, Paper, Scissors" game consisting of multiple rounds and sure chosen attacks.

Robots vs. Dinosaurs

An interactive console application that will have Robots and Dinosaurs fight in a battle. It's a two-player game to the death.

Clock Radio

A Console application representing an alarm clock with the functionality to set the alarm.

Sweepstakes

A backend application that provides the functionality to manage many sweepstakes.
